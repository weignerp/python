Cookies we use

Cookie Desciption
LFR_SESSION_STATE_userId Technical session cookie (expires once the session ends). It belongs to Liferay. It contains the timestamp when the user's session started.
COOKIE_SUPPORT Technical cookie (duration 1 year). It belongs to Liferay. It is used to know if the user admits cookies.
JSESSIONID Technical session cookie (expires once the session ends). It belongs to Liferay. It is used by sites written in JSP to maintain an anonymous user session by the server.
GUEST_LANGUAGE_ID Functional personalization cookie (duration 1 year). It belongs to Liferay. It remembers the user's language preference.
USER_CONSENT_CONFIGURED Technical cookie. Stores the state of the user consent with cookies.
CONSENT_TYPE_FUNCTIONAL Technical cookie. Stores the state of the user consent with the use of functional cookies.
CONSENT_TYPE_NECESSARY Technical cookie. Stores the state of the user consent with the user of necessary cookies.
CONSENT_TYPE_PERFORMANCE Technical cookie. Stores the state of the user consent with the user of performance cookies.
CONSENT_TYPE_PERSONALIZATION Technical cookie. Stores the state of the user consent with the user of personalization cookies.

Third party cookies

Cookie Desciption
